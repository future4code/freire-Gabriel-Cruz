Exercícios
IMPORTANTE 1) Crie uma branch a partir da branch master para trabalhar no exercício de hoje. O nome da branch de hoje deve ser: css-bugfixes 2) Dentro da pasta do módulo atual, crie uma pasta chamada css-bugfixes para trabalhar no exercício de hoje
Treino de JavaScript
Enunciado
No exercício de hoje, você vai treinar a depuração da melhor maneira possível: corrigindo Bugs!
Senta que lá vem história
No mercado de trabalho, temos o costume de chamar nossas tarefas do dia a dia de "issues" (já deu pra reparar que devs não largam o inglês né? ). Para tentar aproximar vocês um pouco deste dia a dia, estamos reportando as issues de uma forma parecida com o como seria em uma empresa.
Para simular ainda mais esta situação, vamos pedir que, para cada issue finalizada você dê, ao menos, 1 commit, explicando qual bug você arrumou e, de uma maneira bem sucinta, como foi resolvido.
Exemplo de commits
Os arquivos para você utilizar encontram-se neste zip:
exercicio-bugfixes.zip

Instruções pro Zip
As issues estão descritas abaixo, resolva-as na ordem em que estão.
1 - [Tela Comprar] Botão "Voltar" não está funcionando
2 - [Tela Home] Botão "Comprar" está no lugar errado
3 - [Tela Home] A mensagem da seção azul está muito próxima do cabeçalho
4 - [Tela Home] Formatação da mensagem da seção azul está errada
5 - [Tela Comprar] O texto da tabela está com a formatação errada
6 - [Tela Comprar] As imagens da tabela não estão iguais
7 - [Tela Comprar] Ao lado do campo de busca, deveria ter um ícone com uma imagem de lupa
8 - [Tela Comprar] Colocar efeitos de clique e sobreposição nos botões
9 - [Tela Comprar] Arrumar borda do campo de texto
10 - [Tela Comprar] Arrumar a fonte de texto
11 - [Tela Comprar] Centralizar a tabela
